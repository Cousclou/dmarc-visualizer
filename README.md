# dmarc-visualizer

Analyse and visualize DMARC results using open-source tools.

* [parsedmarc](https://github.com/domainaware/parsedmarc) for parsing DMARC reports,
* [Elasticsearch](https://www.elastic.co/) to store aggregated data.
* [Grafana](https://grafana.com/) to visualize the aggregated reports.
* [Grafana Dashboard](https://github.com/bhozar/grafana-dashboards) by bhozar
* [dmarc-visualizer](https://github.com/debricked/dmarc-visualizer) source by Debricked
* [dmarc-visualizer](https://wiki.cous.re/selfhosted/dmarc-visualizer) m'a procédure d'installation
